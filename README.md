# UoL-GORM
Final project for CM3070. 

## Controls
- Movement - WASD
- Attack - Left-click
- Block - Right-click
- Cast spell - F
- Open chest - E
- Select spell - 1, 2, 3
- Pause menu - F1
- Stats menu - F2
- Spells menu - F3

## Known Bugs
- Gaps between some walls
- Enemies can hit player out of walls
- Spamming stat / spell screen buttons during level loading can softlock game
